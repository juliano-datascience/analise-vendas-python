# 📊 Análise de Vendas com Python

Projeto de análise de dados utilizando Python para explorar um conjunto de vendas e gerar insights de negócio.

---

## 🎯 Objetivo

Analisar dados de vendas para identificar:

- Produtos mais vendidos  
- Produtos com maior faturamento  
- Distribuição de vendas  
- Padrões de consumo  

---

## 🛠️ Ferramentas Utilizadas

- Python  
- Pandas  
- Plotly  
- Google Colab  

---

## 📂 Dataset

O dataset contém informações de vendas como:

- Produto  
- Quantidade vendida  
- Valor unitário  
- Data da venda  
- Localização  

---

## 📈 Principais Análises

- Cálculo de faturamento total  
- Ranking de produtos mais vendidos  
- Ranking de faturamento por produto  
- Curva ABC de produtos  
- Visualização de dados com gráficos  

---

## 💡 Insights

- Notebook e PC são os produtos com maior faturamento  
- Produtos de menor valor têm maior volume de vendas  
- A maior parte do faturamento está concentrada em poucos produtos (Curva ABC)  

---

## 🚀 Resultado

O projeto ajuda a entender o comportamento de vendas e apoiar decisões estratégicas baseadas em dados.
